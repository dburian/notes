# Ships log

Ships log is a book containing notes and records with which a navigator should
be able to **relatively accurately estimate the current ship's position at any
given time**.

The minimum information for recreational vessels according to the [Complete
Yachtmaster](https://www.amazon.com/Complete-Yachtmaster-Sailing-Seamanship-Navigation/dp/1472982983) is to record:

- local time
- log -- number of miles the ship has sailed
- kormidelní kurz (heading?)
- COD -- course over ground
- current weather
- position
- a large space for notes

It is recommended to make a record every hour or change of course.
